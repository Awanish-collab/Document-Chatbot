# Document Q&A Chatbot

A Q&A Chatbot built with Python and GenAI libraries, designed to handle multiple file formats such as PDF, DOCX, PPT, MP4, and audio files. This chatbot extracts information from uploaded documents and provides answers to queries using GenAI models from Hugging Face.
